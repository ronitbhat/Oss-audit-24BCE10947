# OSS Audit Project — Git

An Open Source Software Audit project focused on **Git**, demonstrating Linux system analysis and automation using shell scripting.

---

## Project Description

This project explores **Git** from both theoretical and practical perspectives within a Linux environment. It highlights:

- Open-source philosophy and licensing  
- Real-world Linux system interaction  
- Automation using shell scripting  

The goal is to understand how open-source tools like Git function at a system level and how they can be used effectively in development workflows.

## Scripts Included

This repository contains **5 shell scripts**, each focusing on a specific system-level task:

---

### 1. System Identity Report
Displays detailed system information including:
- OS distribution  
- Kernel version  
- User details  
- System uptime  
- Current date and time  

---

### 2. Git Installation Checker
- Verifies whether Git is installed  
- Displays installed version  
- Confirms system configuration  

---

### 3. Directory Auditor
- Checks important system directories (`/etc`, `/usr/bin`, `/home`)  
- Displays permissions and ownership  
- Ensures directory availability  

---

### 4. System Log Analyzer
- Analyzes system logs  
- Counts error and warning entries  
- Displays recent critical logs  

---

### 5. Open Source Manifesto
- Prints core principles of open-source software  
- Highlights freedom, transparency, and collaboration
